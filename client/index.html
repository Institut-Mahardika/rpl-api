Untuk dosen — menjalankan server

⚙️ Cara Menjalankan Server Lokal

# 1. Clone repository
$ git clone https://github.com/institut-mahardika/rpl-api.git
$ cd rpl-api/server
# 2. Install dependencies (sekali saja)
$ pip install -r requirements.txt
# 3. Jalankan server
$ uvicorn main:app --host 0.0.0.0 --port 3000 --reload
Dokumentasi API
http://<IP>:3000/docs
Admin Panel
http://<IP>:3000/admin/ui
Cek IP laptop (macOS/Linux)
ifconfig | grep inet
Cek IP laptop (Windows)
ipconfig
